## 👋 Hi, I'm Kranthi

Founder & CEO of **[NarneTech Software Solutions](https://narnetech.com)**, based in Vijayawada, India. I'm a full-stack engineer who likes building things end-to-end — from **C# / ASP.NET Core** web platforms and ERP systems to **ESP32 firmware** humming away on a factory floor.

These days most of my time goes into **industrial IoT**, **energy intelligence**, and bringing **AI agents** into real ERP workflows.

---

## 🚧 What I'm Working On

- 🏭 **PlantPulse** — an AI-powered industrial EMS + CMMS platform
- 🛰️ **ESP32-S3 Modbus IoT gateways** — dual-core FreeRTOS, MQTT, OTA
- 🤖 **AI + RAG inside ERP** — Semantic Kernel, Qdrant, GPT-4o-mini
- 🌐 **Full-stack ASP.NET Core platforms** for SMEs across Andhra Pradesh
